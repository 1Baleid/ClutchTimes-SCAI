# ClutchTimes-SCAI

A proof-of-concept suite showcasing three computer-vision modules for soccer broadcast automation:

| Module                                      | Functionality                                          |
|---------------------------------------------|--------------------------------------------------------|
| **BallTrackerModel**                        | Detects & tracks the soccer ball in 2D broadcast video |
| **CameraOptimizationBasedOnCoordinates**     | Generates camera control commands from object coordinates |
| **FaceRecognition**                         | Detects & recognizes faces of players/staff            |
